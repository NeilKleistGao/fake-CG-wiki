    光线追踪简介

    当你凝视深渊的时候，深渊也在凝视你 —— 弗里德里希·威廉·尼采

    本章主要介绍光线追踪的基本原理。

    光栅化存在的最严重的问题是：它本身并不模拟光线的运动，所以对于诸如全局光照这样的计算只能通过近似完成，而且效果可能并不理想。 为了解决这个问题，我们需要引入光线追踪，从而真正地模拟光线的运动。
